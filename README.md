# AI Catalyst — 직원 발표용 파일럿 데모

## 데모 실행 방법

**가장 쉬운 방법**: `index.html` 파일을 브라우저에서 더블클릭으로 열면 바로 실행된다.

(별도 서버 불필요. 모든 자산은 정적 HTML/CSS/JS로 동작한다.)

## 페이지 구성

| 파일 | 화면 |
|------|------|
| `index.html` | 메인 랜딩 (Hero, 소개, 4단계 안내, 인기 강의, 두 가지 길, CTA) |
| `courses.html` | 전체 강의 목록 + 카테고리 필터 |
| `course-detail.html` | 강의 상세 (강사 프로필, 커리큘럼, 신청 사이드바) |
| `mypage.html` | 마이페이지 (수강내역, 환급신청, 평점작성 탭) |
| `instructor.html` | 강의 개설 신청 폼 |

URL 파라미터로 강의 전환: `course-detail.html?id=3`

## 데모 인터랙션

- 모든 페이지 모바일 반응형 (640px / 980px 브레이크포인트)
- 강의 카드 클릭 → 상세 페이지로 이동
- "강의 신청하기" 버튼 → 신청 완료 모달 표시
- 마이페이지 탭 전환 (수강내역 / 환급 / 평점)
- 강의 개설 폼 제출 → 신청 완료 모달
- 카테고리 필터 칩으로 강의 즉시 필터링

## 디자인 가이드

- Apple 디자인 시스템 기반: SF Pro / Apple SD Gothic Neo, 큰 여백, 둥근 코너, 미니멀 컬러
- 메인 컬러: 흑백(#1d1d1f / #f5f5f7) + 액센트 블루(#0071e3)
- 디지털다임 로고: `assets/logo.svg` (텍스트 워드마크)
  → 실제 회사 공식 로고 파일을 받으면 이 경로의 SVG/PNG로 교체만 하면 된다.


## 데모용 샘플 데이터

8개 강의가 등록되어 있다 (개발/디자인/마케팅/기획/전략 카테고리 분포):
1. ChatGPT 이메일 자동화 — 김지훈
2. Midjourney 광고 시안 — 박서연
3. Claude Code 코드 리뷰 — 이민호
4. Notion AI 회의록 — 정현우
5. Cursor IDE 개발 속도 — 강민정
6. Perplexity 시장 리서치 — 윤재호
7. Gamma 임원 보고용 PPT — 한수진
8. Suno 사내 BGM 제작 — 최도윤

마이페이지의 사용자는 "홍길동 / 전략기획실 / 사번 D2-2019-0042"로 설정되어 있다.

## 발표 시연 시나리오 추천 흐름

1. **index.html** 열기 → Hero 영역과 "AI Catalyst란?" 까지 스크롤하며 컨셉 설명
2. "참여 방법 4단계" → 운영 흐름 설명
3. "지금 인기있는 강의" 카드 클릭 → **course-detail.html**로 이동 → 커리큘럼·사이드바 시연
4. "강의 신청하기" 버튼 클릭 → 신청 완료 모달 시연
5. 상단 메뉴에서 **마이페이지** → 환급/평점 탭 전환 시연
6. **instructor.html**로 이동 → 강사 입장 폼 시연
7. **courses.html**로 이동 → 카테고리 필터 시연
8. 모바일 시뮬레이션 (브라우저 개발자 도구 모바일 뷰)으로 반응형 시연
